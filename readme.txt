DCCODE WP User Avatar makes it easy to add photo to user's profile.

== Description ==

This plugin allows you and your users to use local avatar.

 * Easy to upload user's avatar

== Installation ==

1. Upload the folder 'dccode-user-avatar' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What image extensions are allowed to use as avatar?

jpg | gif | png extensions are allowed.
